local dev commands:

  > `make dash`      -- run local dev for dashboard site
  > `make site`      -- run local dev for main site
  > `make storybook` -- run local dev for components/storybook
  > `make docs`      -- run local dev for docs site
  > `make all`       -- run local dev for all 3 simultaneously
  > `make clean`     -- remove all caches, good when troubleshooting weird stuff
  > `make ts-watch`  -- typescript check all subdirs, watch mode

quality/ci commands:

  > `make test`       -- run all unit tests
  > `make test-watch` -- run all unit tests, watch mode
  > `make format`     -- run prettier on all files
  > `make lint`       -- run eslint
  > `make lint-fix`   -- run eslint, auto-fixing where possible
  > `make ts-check`   -- typescript check all subdirs
  > `make check`      -- test for prettier, lint, typescript errors
  > `make fix`        -- run lint fix and prettier format

scaffold commands:

  > `make component [@dash|@main|@shared] [name]` -- scaffold files for a new react component
